# 01-go-basics：Go 语言基础语法模块

## 模块说明

本目录是成都信息工程大学（CUIT）Golang 教学体系的入门核心，包含 12 个基础语法 Demo，覆盖从程序入口到函数、指针、常量的完整基础知识点。所有代码均附带详细中文注释，遵循 Go 语言编码规范，适合零基础学习者循序渐进掌握语法，也可作为教师课堂演示、学生课后练习的标准化素材。

## 学习路径（推荐顺序）

按以下顺序学习，符合编程语言学习的逻辑，避免知识断层：

001-demo-HelloWorld（程序入口与基础打印）

002-demo-Variable（变量声明与使用）

003-demo-Scope（变量作用域与内存存储）

004-demo-NumberTypes（数值类型与取值范围）

005-demo-Bool（布尔类型与逻辑运算）

006-demo-String（字符串操作与编码特性）

007-demo-TypeConversion（显式类型转换规则）

008-demo-ControlFlow（条件分支语句）

009-demo-Loop（循环与流程控制）

010-demo-Function（函数定义与闭包）

011-demo-Pointer（指针与内存地址）

012-demo-Const（常量与枚举器）

## 运行与调试

### 1. 单 Demo 运行

进入目标 Demo 目录，执行go run命令：

```
# 示例：运行HelloWorld
cd 001-demo-HelloWorld
go run hello_world.go
```

### 2. 常见问题解决

- **"undefined: xxx" 报错**：检查包导入完整性、变量 / 函数名大小写（Go 区分大小写）；

- **类型转换错误**：确认转换类型是否有相同底层类型（如 bool 不可直接转 int）；

- **闭包变量值异常**：通过函数参数捕获循环变量当前值，避免延迟绑定问题。

## 学习建议

**实操优先**：每个 Demo 先理解注释，再修改代码（如调整循环条件、变量类型），观察结果变化；

**重点突破**：优先掌握变量作用域、指针传值、闭包、字符串拼接性能等易混淆知识点；
